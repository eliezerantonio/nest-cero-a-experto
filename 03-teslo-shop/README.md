# Teslo API

1. Clone Poject
2. ``` yarn install ```
3. clone   and rename archive ``` .env.template ``` to ```.env```
4. Change environment variables
5. Raise the BD

```
docker compose up
```

6. Start the app

 ```
 yarn start:dev
 ```
